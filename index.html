¿Qué es Git?

Git es un sistema de control de versiones que permite llevar un registro de los cambios en tu código. Ayuda a colaborar en proyectos, regresar a versiones anteriores y trabajar de forma organizada en equipo.

¿Qué hace?

Con Git puedes guardar el historial de tus proyectos, crear ramas para probar nuevas ideas sin afectar el trabajo principal y compartir tu código fácilmente en plataformas como GitHub.
